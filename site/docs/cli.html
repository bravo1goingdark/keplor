CLI Reference

All commands available in the keplor binary.

keplor run

Start the HTTP ingestion server.

$ keplor run [OPTIONS]
OptionDefaultDescription
-c, --config <PATH>keplor.tomlPath to TOML config file

If the config file doesn't exist, defaults are used and a warning is logged. The server binds to 0.0.0.0:8080 and creates keplor.db in the current directory.

keplor migrate

Apply database migrations without starting the server. Useful for upgrading the schema before deployment.

$ keplor migrate [OPTIONS]
OptionDefaultDescription
-d, --db <PATH>keplor.dbPath to the SQLite database
 
$ keplor migrate --db /var/lib/keplor/keplor.db
migrations applied to /var/lib/keplor/keplor.db

keplor query

Query stored events from the command line.

$ keplor query [OPTIONS]
OptionDefaultDescription
--user-id <ID>Filter by user ID
--model <NAME>Filter by model name
--provider <KEY>Filter by provider
--source <NAME>Filter by ingestion source
--limit <N>20Maximum results
-d, --db <PATH>keplor.dbPath to the SQLite database
 
$ keplor query --provider openai --limit 5
ID                           PROVIDER       MODEL                   TOKENS       COST ($)
--------------------------------------------------------------------------------------------

01J5XPBN7K1D2M6R...         openai         gpt-4o-mini                340     0.00012750

2 event(s)

keplor stats

Print storage statistics: event count, blob count, compression ratios.

$ keplor stats [OPTIONS]
OptionDefaultDescription
-d, --db <PATH>keplor.dbPath to the SQLite database
 
$ keplor stats
=== Keplor Storage Statistics ===
Database:             keplor.db
Total events:         12,847
Unique blobs:         3,219
Compressed size:      1,847,293 bytes
Uncompressed size:    24,671,840 bytes
Compression ratio:    13.4x

keplor gc

Delete events older than a threshold and clean up orphaned blobs.

$ keplor gc --older-than-days <DAYS> [OPTIONS]
OptionRequiredDescription
--older-than-days <N>yesDelete events older than N days
-d, --db <PATH>noPath to the SQLite database (default: keplor.db)
 
$ keplor gc --older-than-days 30
GC complete: deleted 4,291 events, 847 orphaned blobs (cutoff: 30 days ago)

Run periodically via cron to prevent unbounded database growth:

# crontab -e
0 3 * * * /usr/local/bin/keplor gc --older-than-days 90 --db /var/lib/keplor/keplor.db
