API Reference

Base URL: http://localhost:8080

Authentication

When API keys are configured, all /v1/* endpoints require a Bearer token:

Authorization: Bearer your-api-key-here

When no keys are configured (default), authentication is disabled. The /health and /metrics endpoints are always public.

POST /v1/events

Ingest a single LLM event. Waits for durable storage (batched write flush). Times out after 10 seconds.

Request body

FieldTypeRequiredDescription
modelstringyesModel name (e.g. "gpt-4o")
providerstringyesProvider key (e.g. "openai", "anthropic")
usage.input_tokensu32noInput/prompt tokens
usage.output_tokensu32noOutput/completion tokens
usage.cache_read_input_tokensu32noCached prompt tokens read
usage.cache_creation_input_tokensu32noTokens written to cache
usage.reasoning_tokensu32noReasoning/thinking tokens
latency.ttft_msu32noTime to first token (ms)
latency.total_msu32noEnd-to-end latency (ms)
timestampstring | i64noISO 8601 or epoch nanoseconds. Defaults to now.
user_idstringnoCaller-provided user identifier
api_key_idstringnoAPI key identifier for attribution
org_idstringnoOrganization for cost rollups
sourcestringnoSending system (e.g. "litellm")
endpointstringnoAPI path (default "/v1/chat/completions")
http_statusu16noUpstream HTTP status code
flags.streamingboolnoWhether response was streamed
flags.tool_callsboolnoWhether response had tool calls
request_bodyobjectnoFull request JSON (stored compressed)
response_bodyobjectnoFull response JSON (stored compressed)
error.kindstringnorate_limited, auth_failed, context_length_exceeded, etc.
error.messagestringnoHuman-readable error description

Response 201 Created

{
  "id": "01J5XQKR4M2E3N8V7P6Y1WDCBA",
  "cost_nanodollars": 6250000,
  "model": "gpt-4o",
  "provider": "openai"
}

Cost is auto-computed from the bundled LiteLLM pricing catalog. 1 nanodollar = $0.000000001.

POST /v1/events/batch

Ingest up to 10,000 events in a single request. Uses fire-and-forget writes for throughput.

Request body

{
  "events": [
    { "model": "gpt-4o", "provider": "openai", ... },
    { "model": "claude-sonnet-4-20250514", "provider": "anthropic", ... }
  ]
}

Response 201 Created or 207 Multi-Status

{
  "results": [
    { "id": "01J5X...", "cost_nanodollars": 100, "model": "gpt-4o", "provider": "openai" },
    { "error": "validation: model must not be empty" }
  ],
  "accepted": 1,
  "rejected": 1
}

Returns 201 when all events succeed, 207 when some fail. Maximum batch size: 10,000 events.

GET /v1/events

Query stored events with filters and cursor-based pagination.

Query parameters

ParameterTypeDescription
user_idstringFilter by user ID
api_key_idstringFilter by API key ID
modelstringFilter by model name
providerstringFilter by provider
sourcestringFilter by ingestion source
fromi64Events after this epoch nanosecond
toi64Events before this epoch nanosecond
limitu32Max results (default 50, max 1000)
cursori64Pagination cursor from previous response

Response 200 OK

{
  "events": [
    {
      "id": "01J5XQKR...",
      "timestamp": 1700000000000000000,
      "model": "gpt-4o",
      "provider": "openai",
      "usage": {
        "input_tokens": 500,
        "output_tokens": 200,
        "cache_read_input_tokens": 0,
        "reasoning_tokens": 0
      },
      "cost_nanodollars": 6250000,
      "latency_total_ms": 1200,
      "latency_ttft_ms": 180,
      "http_status": 200,
      "source": "my-app",
      "user_id": "alice",
      "endpoint": "/v1/chat/completions",
      "streaming": false
    }
  ],
  "cursor": 1700000000000000000,
  "has_more": false
}

To paginate, pass the cursor value from the response as a query parameter in the next request.

GET /health

Liveness probe with database connectivity check. Always public (no auth required).

{"status":"ok","version":"0.1.0","db":"connected"}

Returns 200 when healthy, 503 when the database is unreachable.

GET /metrics

Prometheus metrics in text exposition format. Always public.

# HELP keplor_events_ingested_total Total events ingested
# TYPE keplor_events_ingested_total counter
keplor_events_ingested_total{provider="openai"} 42
keplor_events_ingested_total{provider="anthropic"} 18

Error responses

All errors return a JSON body with an error field:

{"error": "validation: model must not be empty"}
StatusWhen
400Validation error, bad JSON, invalid timestamp
401Missing or invalid API key
422Unknown provider
500Internal error (storage failure, timeout)
