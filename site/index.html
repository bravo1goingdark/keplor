What you get

Everything you need to understand your LLM spend and behavior, nothing you don't.

cost_nanodollars

Nanodollar cost tracking

Auto-computed from the LiteLLM pricing catalog. Per-user, per-key, per-model attribution. Daily rollups.

request_body + response_body

Full-body capture

Request and response stored with zstd compression and content-addressed deduplication. System prompts are stored once.

sha256 + refcount

Content-addressed dedup

Identical system prompts, tools, and schemas are stored once with reference counting. Trained zstd dictionaries for 5x compression.

How it works

1

Instrument your gateway

Point your LiteLLM proxy, custom gateway, or application code at Keplor's ingestion endpoint. One HTTP POST per LLM call.

2

Keplor processes and stores

Events are validated, normalized, cost-computed, compressed, and deduplicated into a local SQLite database. Batched writes for throughput.

3

Query and analyze

Use the REST API or CLI to query events, check costs, and export data. Prometheus metrics for real-time dashboards.

Integrate in minutes

One HTTP call. Any language.

curl -X POST http://localhost:8080/v1/events \
  -H "Authorization: Bearer $KEPLOR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "provider": "openai",
    "usage": {
      "input_tokens": 1200,
      "output_tokens": 350
    },
    "latency": {
      "ttft_ms": 180,
      "total_ms": 2400
    },
    "user_id": "user_alice",
    "source": "api-gateway"
  }'
import requests

resp = requests.post(
    "http://localhost:8080/v1/events",
    headers={"Authorization": f"Bearer {api_key}"},
    json={
        "model": "gpt-4o",
        "provider": "openai",
        "usage": {
            "input_tokens": 1200,
            "output_tokens": 350,
        },
        "user_id": "user_alice",
        "source": "api-gateway",
    },
)
# {"id": "01J5X...", "cost_nanodollars": 6250000, ...}
const resp = await fetch("http://localhost:8080/v1/events", {

    "Content-Type": "application/json",

  body: JSON.stringify({
    model: "gpt-4o",
    provider: "openai",
    usage: { input_tokens: 1200, output_tokens: 350 },
    user_id: "user_alice",

  }),
});

const data = await resp.json();
